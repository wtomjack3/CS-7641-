### Directory structure:
  ./ : Python Code Files
  .Data/: Data Sets used to generate experiments
  ./Documents: Instructions and reports for experiments
  ./plots: Plotted images of experiments

### Code Dependencies:
  Python3
  numpy
  scikit-learn
  matplotlib
  pandas

### Instructions for running experiments
  -Pull entire repository from GITHUB at this address
  -Open command prompt, navigate to appropriate SupervisedLearning directory. 
  -Execute command 'python main.py'
  -Running the main.py file will generate all graphs/charts and output data either in terminal or sub-directory "plots"
